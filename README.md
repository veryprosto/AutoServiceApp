# 🚗 AutoService Management System (MVP)

Полноценная система управления автосервисом, построенная на стеке **.NET 9** и **Vanilla JavaScript**. Проект реализует полный цикл работы с клиентами: от регистрации автомобиля до формирования финансовых отчетов по закрытым заказам.

## 🚀 Основные возможности

*   **Управление клиентами и автопарком:** Регистрация владельцев и привязка автомобилей к каждому профилю.
*   **Система заказов (Work Orders):** Создание заявок, отслеживание этапов (Новый -> В работе -> Завершен) и учет стоимости запчастей/работ.
*   **Безопасность и роли:** Авторизация на базе **JWT**. Разделение прав:
    *   `Owner` — полный доступ к финансам и персоналу.
    *   `Master` — управление заказами.
    *   `Mechanic` — просмотр задач.
*   **Финансовая аналитика:** Расчет выручки и статистика выполненных заказов.
*   **Интерактивный UI:** SPA-интерфейс на чистом JS с сохранением состояния через URL.

## 🛠 Технологический стек

*   **Backend:** .NET 9 Web API
*   **Database:** SQLite + Entity Framework Core
*   **Security:** JWT Bearer Authentication
*   **Frontend:** HTML5, CSS3, JavaScript (Fetch API)

## 📦 Как запустить проект локально

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   ```
2. **Перейдите в папку проекта:**
   ```bash
   cd AutoServiceApp
   ```
3. **Обновите базу данных:**
   ```bash
   dotnet ef database update
   ```
4. **Запустите приложение:**
   ```bash
   dotnet watch
   ```
