🚗 AutoService Management System (MVP)
Полноценная система управления автосервисом, построенная на стеке .NET 9 и Vanilla JavaScript. Проект реализует полный цикл работы с клиентами: от регистрации автомобиля до формирования финансовых отчетов по закрытым заказам.
🚀 Основные возможности
Управление клиентами и автопарком: Регистрация владельцев и привязка неограниченного количества автомобилей к каждому профилю.
Система заказов (Work Orders): Создание заявок на ремонт, отслеживание этапов (Новый -> В работе -> Завершен) и учет стоимости запчастей/работ.
Безопасность и роли: Полноценная авторизация на базе JWT (JSON Web Tokens). Разделение прав доступа для ролей:
Owner — полный доступ к финансам, отчетам и управлению персоналом.
Master — управление заказами и клиентами.
Mechanic — просмотр задач.
Финансовая аналитика: Автоматический расчет выручки, разделение доходов на услуги и запчасти, статистика по количеству выполненных заказов.
Интерактивный UI: Одностраничный интерфейс (SPA-style) на чистом JS с сохранением состояния через URL-параметры.
🛠 Технологический стек
Backend: .NET 9 Web API
Database: SQLite + Entity Framework Core (Code First)
Security: Microsoft.AspNetCore.Authentication.JwtBearer
Frontend: HTML5, CSS3 (Custom Styles), JavaScript (Fetch API, LocalStorage)
API Documentation: Swagger / OpenAPI 3.0
📦 Как запустить проект локально
Клонируйте репозиторий:
bash
git clone https://github.com
Используйте код с осторожностью.
Перейдите в папку проекта:
bash
cd AutoServiceApp
Используйте код с осторожностью.
Обновите базу данных:
bash
dotnet ef database update
Используйте код с осторожностью.
Запустите приложение:
bash
dotnet watch
Используйте код с осторожностью.
Откройте в браузере:
http://localhost:5032 (Фронтенд откроется автоматически благодаря настроенному wwwroot).
